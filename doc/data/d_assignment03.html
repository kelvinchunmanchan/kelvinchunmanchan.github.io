Downloading data

I first use the load_variable() command to search the code list. I select two variables for analysis: Chinese, which is the total number of Chinese excluding Taiwanese, and Gross_rent, which is the median gross rent. All data are at the county level. Then, I use get_acs() to retrieve the data and keep the estimate and margin of error in a wide-format data frame.

vars <- load_variables(2023, "acs5", cache = TRUE) # Check variable list

state_abbr <- "TX"
geo_level  <- "county"   # options: state, county, tract, block group
my_vars    <- c(chinese = "B02015_002", gross_rent = "B25064_001")
year_acs   <- 2023
survey     <- "acs5"

# Use the get_acs function to download the data
acs <- get_acs(
    geography = geo_level,
    variables = my_vars,
    state = state_abbr,
    # county = "Hansford",  (Switch to county results)
    year = year_acs,
    survey = survey,
    geometry = TRUE
)

# Convert table from long to wide for ggplot2
acs_wide <- acs |>
    tidyr::pivot_wider(
        id_cols = c(GEOID, NAME, geometry),
        names_from = variable,
        values_from = c(estimate, moe)
    )

Plot 1

The results show that Chinese populations are mainly concentrated in the metropolitan areas of Texas, including Houston, Austin, San Antonio, and DallasFort Worth. Among all Texan cities, Houston has the highest number of Chinese residents.

ggplot(acs_wide) +
    geom_sf(aes(fill = estimate_chinese), color = NA) +
    scale_fill_viridis_c(name = "Chinese people") +
    labs(title = paste0("ACS ", year_acs, " 5-year: Total: Chinese, except Taiwanese  ", state_abbr, " (", geo_level, ")"),
         caption = "Source: U.S. Census Bureau via tidycensus") +
    theme(plot.title = element_text(hjust = 0, size = 14, face = "bold"), 
          plot.caption = element_text(hjust = 0, face = "italic"), 
          plot.caption.position = "plot", # Relative to plot
    )

Plot 2

The second plot shows the median gross rent. As expected, the metropolitan counties have the highest rents. The heatmap also indicates that the eastern Texas counties generally have higher rents than the less populous western counties.

ggplot(acs_wide) +
    geom_sf(aes(fill = estimate_gross_rent), color = NA) +
    scale_fill_viridis_c(name = "Median rent", direction = -1, option = "plasma") +
    labs(title = paste0("ACS ", year_acs, " 5-year: Median rent  ", state_abbr, " (", geo_level, ")"),
         caption = "Source: U.S. Census Bureau via tidycensus") +
    theme(plot.title = element_text(hjust = 0.5, size = 14, face = "bold"), 
          plot.caption = element_text(hjust = 0, face = "italic"),
